# Catalysis Notes

Use these reminders when interpreting simple DFT post-processing results:

- adsorption energies depend on the reference state and surface model
- a d-band center is a descriptor, not a sufficient explanation of activity
- a barrier from NEB images should be discussed together with the initial and final state energetics
- heuristic ranking is useful for screening, but it is not a replacement for microkinetics or coverage effects
